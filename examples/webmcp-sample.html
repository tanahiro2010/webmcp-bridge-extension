WebMCP Sample Page

これはごく普通の Web ページです。WebMCP Bridge 拡張機能をインストールしていれば、 右上に「WebMCPをインストール」というオーバーレイボタンが表示されるはずです。 これは拡張機能がこのページ上の WebMCP tool を検出した合図です。

このページは WebMCP 仕様 に沿った2種類の tool を公開しています。

  • reserve_hotel — 命令型 (document.modelContext.registerTool())
  • search_hotels — 宣言型 (<form toolname tooldescription>)
